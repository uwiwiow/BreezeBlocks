# BreezeBlocks
## Controls
- left arrow / right arrow
- A / D
  
Or press `m` to activate mouse mode
## Install
```
git clone https://github.com/uwiwiow/BreezeBlocks.git
git submodule init
git submodule update
```

## Build
Linux
```sh
cd <PROJECT_DIR>
mkdir build
cd build
cmake ..
make
```
### Run
`./BreezeBlocks`

## Requires
- SDL2
- SDL2_gfx
